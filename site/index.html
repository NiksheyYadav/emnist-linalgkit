Neural Network from Scratch with LinAlgKit

This repository contains an educational deep-learning implementation using NumPy + LinAlgKit.

To get started locally, install dependencies with pip install -r requirements.txt and run:

  • python train.py --epochs 10 (MNIST)
  • python train_emnist.py --split balanced --epochs 10 (EMNIST)

Full documentation and examples are available in the GitHub repository.
